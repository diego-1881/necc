# The New England Center for Children

Front-end development for The New England Center for Children (NECC) main website.

This project contains the hand-crafted HTML, CSS, and JavaScript used to integrate and build a custom WordPress theme.

## Features

- Responsive multi-page marketing site for NECC
- Program and services sections (Southborough, Consulting & PSS, etc.)
- Blog and single-article layouts
- Reusable components
- Design system playground to test typography, components, and icons

## Technology Stack

- Vite
- Bootstrap 5
- Custom SCSS
- Vanilla JavaScript
- Swiper

## Getting Started

In the project directory, you can run:

### `npm install`

Installs all project dependencies required for development and building.

### `npm run dev`

Starts the Vite development server.

Open http://localhost:8090 to view it in your browser.
